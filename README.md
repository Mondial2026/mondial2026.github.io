# 🏆 Coupe du Monde 2026 · Focus Côte d'Ivoire 🇨🇮🐘

**Site en ligne : [mondial2026.github.io](https://mondial2026.github.io/)**

Landing page de suivi de la Coupe du Monde FIFA 2026 (11 juin – 19 juillet 2026, États-Unis 🇺🇸 / Canada 🇨🇦 / Mexique 🇲🇽, 48 équipes), avec un focus sur les **Éléphants de Côte d'Ivoire** — de retour au Mondial après 12 ans d'absence, champions d'Afrique en titre (CAN 2023).

## 📋 Contenu

| Onglet | Description |
|---|---|
| 🏠 **Accueil** | Compte à rebours, prochains matchs, dernières actualités |
| 🇨🇮 **Côte d'Ivoire** | Matchs des Éléphants, classement du groupe E, chemin vers la finale (3 scénarios) |
| 📅 **Calendrier** | Les 104 matchs : 72 matchs des 12 groupes + phase finale complète (16ᵉˢ → finale) |
| ⚽ **Résultats** | Tous les matchs joués, du match d'ouverture à la finale |
| 📊 **Classements** | Les 12 groupes, calculés automatiquement à partir des scores |

Horaires affichés en **heure GMT (Abidjan)** puis heure de Paris.

## ⚙️ Fonctionnement

- **Page 100 % autonome** : un seul fichier `index.html`, aucune dépendance, aucun framework — ouvrable hors ligne.
- **Mise à jour quotidienne à 08h00 GMT** : scores de la veille, affiches de la phase finale, classements et actualités (tâche automatisée).
- Les données vivent dans le bloc `DONNÉES` du fichier (`GROUP_MATCHES`, `KNOCKOUT`, `NEWS`…) ; le moteur de rendu recalcule classements, badges et conversions horaires automatiquement.
